# influlink.co
한일 인플루언서 플랫폼

## 페이지
- `/` — INFLULINK: 한일 인플루언서 직접 계약 플랫폼
- `/kbeauty-jp/` — KBEAUTY.jp: 일본인 여행객을 위한 K-Beauty 쇼핑 가이드 (일본어)
  - `index.html` — 현지인 추천 / 한국 유행템 / 일본인 인기템 3개 카테고리 + 필터·검색
  - 제품 카드에 한국어 상품명 표기 (탭하면 복사됨 → 매장에서 바로 활용)
  - 쇼핑 팁, 매장용 한국어 회화, FAQ 포함
  - `blog/` — 블로그 (SEO·애드센스용 콘텐츠): 올영세일 공략 / 일본인 인기 TOP10 / 돈키호테 vs 올리브영
  - `privacy.html` — 개인정보처리방침 (애드센스 필수)
  - `contact.html` — 문의 폼 (Netlify Forms, `contact-thanks.html`로 리다이렉트)
  - `assets/site.css` — 블로그·법적 페이지 공용 스타일시트
  - Google AdSense 광고 슬롯 자리 확보 (승인 후 각 HTML 상단의 주석 처리된 스크립트에 발급 ID 입력)
